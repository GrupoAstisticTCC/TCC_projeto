Sobre o projeto

O projeto Astistic como dito anteriormente foi criado para ser o nosso trabalho de conclusão de curso (TCC), onde o tema escolhido foi o autismo

O website foi criado, dentre outras causas, para conscientizar as pessoas sobre o que é o autismo, seu tratamento e as suas características dentre outras muitas informações reelevantes a respeito deste assunto.
